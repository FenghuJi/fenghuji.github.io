关于Jupyter Notebook

简介

Jupyter Notebook是基于网页的用于交互计算的应用程序。其可被应用于全过程计算：开发、文档编写、运行代码和展示结果。——Jupyter Notebook官方介绍

特点

安装和配置

安装过程

安装结束,运行jupyter noterbook就可以在浏览器中打开jupyter的页面了,默认端口是8888,每次新运行都会端口数+1,默认打开你终端当前的位置,比如你终端目录位于~/Coding,那么就会打开Coding目录.

更改默认打开位置

很多情况下我们打开终端就在家目录,为了进入指定的目录,我们还要cd到自己想要写笔记的目录下,比较麻烦,所以需要更改jupyter默认打开目录

在vim打开页面下搜索c.NotebookApp.default_url,去掉前面的#号,然后比如你想打开目录为~/Study/Jupyter,那么就改为/tree/Study/Jupyter
在新终端运行jupyter notebook就会直接打开你想去的目录了
note:这样选择的话就一定要在家目录下运行jupyter notebook,否则会提示404 NOT FOUND~

修改jupyter的主题

jupyter的默认主题我是真的不喜欢,于是需要更改主题.使用pip install jupyterthemes安装主题更改模块,安装过程有点慢,需要等待一会.
装完之后运行jt -l主题列表,结果系统告诉我找不到jt这个命令,于是运行pip show jupyterthemes,然后返回jupyterthemes包在的位置,比如我的是
~/.local/lib/python2.7/site-packages,那我们jt的位置就在/home/hugo/.local/bin
vim ~/.bashrc,然后在最后加上export PATH="/home/hugo/.local/bin:$PATH",然后source ~/.bashrc就行了

我选了chesterish这个主题,输入命令jt -t chesterish就好了,主题有很多,可以在这个页面看看别人分享的主题截图.
