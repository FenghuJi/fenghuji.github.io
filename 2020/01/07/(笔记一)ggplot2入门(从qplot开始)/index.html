(笔记一)ggplot2入门(从qplot开始)

2020.01.07 Hugo Ji R  热度 ℃

简介

qplot()是学习ggplot2时第一个函数，可以帮助我们快速入门ggplot2并且学习一些专用的术语。
可以学习到的有：

  • 如何将变量映射到图形属性（如颜色，大小，形状）之上
  • 如何指定不同的几何对象来创建不同类型的图形，并合并在一张图上
  • 分面的运行，将数据拆分为子集
  • 如何设定基本选项来调整图形的外观
  • qplot()和plot()之间的一些重要区别

2.4 颜色，大小，形状和其他图形属性

在plot中，用户需要将数据的分类变量转化为plot能理解的类型(如”red”,”yellow”,”green”)，qplot可以自动完成并生成图例：

数据较多时，使用`alpha参数设置透明度`,分母表示经过多少次重叠后颜色不透明： ## 2.5 集合对象 通过修改几何对象(geom)，qplot可以画出任何一种类型的图形： - `geom = "point"`,绘制散点图 - `geom = "smooth"`,绘制拟合平滑曲线，并展示曲线和标准误 - `geom = "boxplot"`，绘制箱线图 - `geom = "path"`和`geom = "line"`,在数据点之间连线，探索时间和其他变量的关系 - `geom = "histogram"`，绘制直方图，"density"则为密度图，"freqpoly"为频率多边形 - `geom = "bar"`,绘制条形图 ### 2.5.1 图中添加平滑曲线
使用`se = FALSE` 和 `method`参数选择不同平滑器，`method = "losse"`使用的是局部回归，曲线的平滑程度由span控制，取值范围从0（很不平滑）到1（很平滑）。当数据较多时，内存消耗较大，可以使用`method = "gam",formula = y ~s(x)`来调用mgcv包拟合一个广义可加模型。大数据，使用 `y ~ s(x,bs ="cs")`。
载splines包以使用自然样条：`formula = y ~ ns(x,2)`，其中第二个参数是自由度，自由度越大，曲线波动越大。 ### 2.5.2 多种类型绘图 - 扰动点图，直方图，密度图，条形图 - 时间序列中的线条图和路径图 线条图和路径图常用于可视化时间序列数据，线条图将点从左到右进行连接，路径图则按照点在数据集中的顺序对其连接（线条图等价于数据按x排序然后绘制路径图）。线条图的x轴一般时时间，展示了单个变量随时间变化的顺序，路径图则展示了两个变量随时间联动的情况，时间反映在点的顺序上。使用`ecomomics`数据集展示。
可以看出来失业率和事业时间是高度相关的。 ## 2.6 分面 分面将数据分割成若干子集，然后创建一个图形矩阵，将每一个子集绘制到图形矩阵的窗格中，所有子图采用相同的图形类型，并进行了一定的设计，使得它们之间易于比较。 qplot()默认的分面方法是将图形拆分为若干窗格，可通过形如`row_var ~ col_var`的表达式指定。如果只想指定一行或者一列，可以使用`.`作占位符，如`row_var ~ .`，会创建一个单列多行的图形矩阵。

2.7 其他选项

  • xlim,ylim: x和y轴的显示区间，如xlim = c(0,20)
  • log: log = “x”告诉x轴取对数
  • main: 图形标题
  • xlab,ylab： x和y轴的标签文字

2.8 qplot和plot函数的区别

  • qplot不是泛型反数：不同的R对象传给qplot时，并不会匹配默认的函数调用，关于泛型函数，可以查看这篇文章查看。
  • 一般而言，可以将一个变量传递给感兴趣的图形属性，这样该变量将进行标度转换并显示在图例上。如果想对其赋值可以使用I()函数:colour = I("red")，将点的颜色设为红色。
  • ggplot2的参数更易懂，(colour,shape,size)比基础绘图的(col,pch,ex)更容易记忆理解
  • 基础绘图中可以使用points(),lines()和text()添加元素，在ggplot2中需要添加额外图层
