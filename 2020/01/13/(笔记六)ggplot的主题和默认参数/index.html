(笔记六)ggplot的主题和默认参数

2020.01.13 Hugo Ji R  热度 ℃

这一节介绍了主题设置，修改参数默认值以及将R的图形转化到LATEX，Word以及其它展示或者文字处理软件中去。

1.主题

主题的控制包括标题，坐标轴标签，图例标签等文字调整，以及网格线，背景，轴须得颜色搭配

主题可以通过对许多参数得控制使图形由粗糙变得美观

  • 使用内置主题，图形每个元素得效果保持着视觉得一致性。默认主题为灰色背景，白色网格。
  • 修改内置主题的某些元素。通过调整和组合这些参数，比如文本的大小和颜色，背景，网格线颜色，文本方向等，可以获得自己独有的主题。

内置主题

内置主题有theme_grey()，theme_bw()，theme_classic()，主题设置有两种方式:

  • 全局设置： 使用theme_set(theme_grey())或theme_set(theme_bw())，theme_set()返回先前的主题，可存储以备后用。
  • 局部设置：图形绘制后使用 + theme_grey()的方式等

主题元素和元素函数

主题元素有：

主题元素 类型 描述
axix.line segment 直线和坐标轴
axis.text.x text x轴标签
axis.text.y text y轴标签
axis.ticks segment 轴须标签
axis.title.x text 水平轴标题
axis.title.y text 竖直轴标题
legend.background rect 图例背景
legend.key rect 图例符号
legend.text text 图例标签
legend.title text 图例标题
panel.background rect 面板背景
panel.border rect 面板边界
panel.grid.major line 主网格线
panel.grid.minor line 次网格线
plot.background rect 整个图形背景
plot.title text 图形标题
strip.background rect 分面标签背景
strip.text.x text 水平条纹文本
strip.text.y text 竖直条纹文本

元素(element)有四个类型：文本(text)，线条（lines），矩形（rectangle），空白（blank），功能为：

  • element_text()：绘制标签和标题，控制字体family，face，color，size，hjust，vjust，angle，linehight。
  • element_line()：绘制线条或线段，控制color，size，linetype
  • element_rect()：背景使用的矩形，控制填充颜色，边界颜色，size，linetype
  • element_blank()：空主题，对元素不分配相应的绘图空间，删去不感兴趣的绘图元素，比如color=NA，fill=NA让某些元素不可见。

2.自定义标度和几何对象

为保证新主题的整体连续型，可能需要调整一些标度和几何对象的默认设置，下文中的函数可避免每次添加标度或几何对象时都不得不手动设置参数。

2.1 标度

如果需要改变默认标度的显示方法，要重新定义ggplot2中的相应图形属性标度函数。该函数的命名方法通常是scale_aesthetics_continuous或者scale_aesthetics_discrete，需将函数命名中的aesthetics替换成相应的图形属性，比如color,fill,size等等。

2.2 几何对象和统计变化

类似update_geom_defaults()和update_stat_defaults()，我们也可以自定义几何对象和统计变换。设置后只影响后续绘制的图形而不是所有图形。

3.存储输出

使用``ggsave()`存储图形，有如下参数

  • path:输出路径
  • width,height,scale(宽高，或图形相对屏幕展示的尺寸)
  • dpi(分辨率，默认为300)
